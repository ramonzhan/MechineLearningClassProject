### 第一次作业：感知机算法实现

2021.3.8

要求：手写感知机算法，并在sklearn中的breast_cancer数据集上进行测试。

实现最简易的感知机算法 $y=\text{sign}(\vec{w}\cdot \vec{x} + b)$，损失函数使用误分类点到分类超平面的总距离，使用随机梯度下降方法对模型进行优化。

参考代码见文件夹perceptron（助教自己写的，若有疑问欢迎提出issue）
